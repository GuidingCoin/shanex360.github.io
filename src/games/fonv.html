Fallout: New Vegas

Description

We like to do Speedruns for this game, Mainly focusing on Yes Man Any% Glitchless but we’ve been learning other categories. The best part by far is Randomizer runs, We use a randomizer to make a completely different build so you have a different playstyle and ending each game. The randomizer also tells you which companions to use and what weapons you’re allowed to use.
