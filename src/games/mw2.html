Description

The best FPS game of all time and it’s not even close. Spec ops speedruns and IW4X fragging are the main focus with the occasional customs game night being held if we have enough people. This game is unplayable matchmaking wise on the 360 because Microsoft can’t figure out how to ban cheaters and hackers on a game that is from 2009! So we play it on PC with IW4X when we want to focus on PVP. Coin (the beast who made this website) has included a tutorial of how to get IW4X working if you’re interested in playing with us there and you own the game on Steam.
